# AmazonAPI商品比較検討アプリ

## 一言でいうと

Amazon APIを活用して、複数商品のスペック・価格・評価・レビュー傾向・メリット/デメリットを比較表にまとめ、購入判断を助けるアプリ。

単なる価格比較ではなく、「自分の用途に合う商品を選ぶ」ための比較検討支援に寄せる。

## 対象ユーザー

- Amazonで商品を比較してから買いたい人
- どれを選べばいいか迷う人
- ガジェット・家電・日用品・サプリ・防災用品などを検討する人
- レビューブログ運営者
- アフィリエイトメディア運営者
- 商品選定記事を作るWeb制作会社・メディア運営者

## 課題

- Amazonの商品ページを複数タブで比較するのが面倒
- スペックやレビューが多すぎて判断しにくい
- 価格だけでなく、サイズ・容量・レビュー傾向・用途別の向き不向きを比較したい
- サクラレビューや極端なレビューに惑わされやすい
- 比較記事を作るのに時間がかかる
- 価格変動や在庫状況を追いにくい

## 解決策

Amazon APIから商品情報を取得し、ユーザーが選んだ複数商品を比較表にする。

さらにAIでレビュー傾向や用途別おすすめを要約し、「安さ重視」「品質重視」「初心者向け」「家族向け」などの切り口で提案する。

## MVP機能

- Amazon商品URL / ASIN登録
- 複数商品の比較リスト作成
- 商品名・画像・価格・評価・レビュー数表示
- スペック比較表
- メリット / デメリットメモ
- 用途別おすすめコメント
- お気に入り比較リスト保存
- Amazonアフィリエイトリンク生成
- 共有用比較ページ

## 比較項目案

- 商品名
- ブランド
- 画像
- 価格
- セール有無
- 評価
- レビュー数
- サイズ
- 重量
- 容量
- カラー
- 型番
- 発売日
- 在庫状況
- 配送条件
- 保証
- 返品可否
- カテゴリ別スペック

## AI活用

- レビュー要約
- 良い口コミ / 悪い口コミの傾向抽出
- 商品ごとの向いている人・向いていない人整理
- 比較表の自動生成
- 購入判断コメント生成
- 商品選定記事の下書き生成
- 類似商品のグルーピング
- 目的別ランキング生成

## ユースケース

### 一般ユーザー向け

- 候補商品を3〜5個入れる
- 比較表を見る
- 自分に合う商品を選ぶ
- Amazonで購入する

### メディア運営者向け

- Amazon商品を複数登録
- 比較表とレビュー要約を自動生成
- 記事に埋め込む
- アフィリエイト導線を作る

### 生活系アプリ連携

- 日用品ストックチェッカーから補充候補を比較
- サプリ手帳からサプリ補充候補を比較
- 防災用品専門店の比較記事を作る

- ガジェット
- 家電

- 防災用品

- ペット用品

- メディア運営者向け月額 2,980円〜9,800円
- 比較表埋め込み機能
- 商品選定記事生成ツール
- 広告

- Amazon Product Advertising API
- ASIN管理

- 比較表生成
- AI要約
- 共有URL
- アフィリエイトタグ管理

## 重要な設計ポイント

- Amazon APIの利用規約を厳守する
- 価格・在庫情報は古くなりやすいため更新日時を表示する
- APIで取得したデータの保存期間・表示ルールに注意する
- レビュー本文の扱いは規約確認が必要
- アフィリエイトリンクであることを明示する
- サクラレビュー判定のような断定表現は慎重にする
- 最初はASIN入力型にして、検索機能は第2フェーズでもよい

## 開発難易度

中〜高。商品比較表そのものは作りやすいが、Amazon APIの審査・利用制限・データ更新・規約遵守が重要。

AI要約や比較記事生成まで入れると価値は出るが、規約とデータ取得範囲を確認する必要がある。

## MVPの切り方

1. ASIN / 商品URL登録
2. 商品情報取得
3. 比較リスト作成
4. 比較表表示
5. AIによる用途別コメント
6. アフィリエイトリンク生成
7. 共有URL作成

## 第2フェーズ

- Amazon内商品検索
- 価格変動通知
- 在庫復活通知
- 比較記事自動生成
- ブログ埋め込みウィジェット
- カテゴリ別テンプレート
- 楽天 / Yahoo!ショッピング連携
- Keepa等の価格履歴データ連携検討

## 差別化ポイント

- 価格だけでなく、用途別の向き不向きまで整理する
- 比較表を簡単に作れる
- メディア運営者向けに記事化しやすい
- 生活系アプリや補充管理と連携できる
- Amazonアフィリエイトと相性がよい

## リスク

- Amazon APIの利用制限
- API審査・アソシエイト審査
- 価格・在庫データの鮮度
- レビュー要約の規約リスク
- アフィリエイト収益単価が低い可能性
- 競合が多い
