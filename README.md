# ObsidianBox

ネタ帳とかいろいろ。

Obsidian用のナレッジベースとして、ビジネスネタ・プロジェクトメモ・Claude Code / Codex向け仕様書を管理するリポジトリです。

## フォルダ構成

```txt
00_Inbox/              思いつき・未整理メモ
02_Business-Ideas/     ビジネスネタ
03_Projects/           実プロジェクト別メモ
04_Specs/              Claude Code / Codex向け仕様書
05_Research/           調査メモ
06_Decisions/          決定事項
07_Tasks/              タスク
08_Templates/          Obsidianテンプレート
```

## 運用方針

- ビジネスネタは `02_Business-Ideas/` に1ネタ1ファイルで保存
- 実装指示に育ったものは `04_Specs/` に切り出す
- 決定事項は `06_Decisions/` に残す
- Claude Code / Codexに渡す前提の仕様は、背景・目的・MVP・画面・DB・受け入れ条件まで書く
